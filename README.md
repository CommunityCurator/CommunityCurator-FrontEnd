# CommunityCurator-FrontEnd

A React.js front-end utilizing Material UI and Tailwind CSS

# Requirements
- [Node.js & npm](https://nodejs.org/en/)

# Running Commands
- npm install
- npm install --save react-tinder-card --legacy-peer-deps
- npm start
